# Bengaluru Nexus Digital Twin: Admin & Citizen Portals

## Domain-Driven Architecture
The project is organized into modular domains to ensure strict separation of concerns and high-fidelity simulation performance.

### 🏛️ Admin Command Center
- **Strategic Sector Management**: 6 key sectors (Strategy, Directives, Builder, Crisis, Social, Reports).
- **Gemma-4 Intelligence**: Local AI advisor for real-time urban policy reporting.
- **Luminous Command UI**: High-contrast light-themed interface for professional operations.

### 👥 Citizen Hub
- **Real-time Directives**: Receive live broadcasts from the Admin Command Center.
- **Spatial Reporting**: GPS-tagged grievance submission with automated tracking.
- **Urban Health Metrics**: AQI and vegetation monitoring.

### ⚙️ Command Core
- **Express / Ollama Node**: Unified API for geospatial analysis and LLM advisory.
- **PostGIS Synchronization**: Spatial database for accurate infrastructure modeling.
